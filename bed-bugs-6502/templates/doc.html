/api/declaration post

post

Create a new declaration.

post /api/declaration

Create a new declaration.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • first_name: required(string)
  • last_name: required(string)
  • address: required(string)
  • inspection_date: required(date-only)
  • neighborhood: required(string)
  • district: required(string)
  • description: (string)

Example:

{
  "first_name": "John",
  "last_name": "Carpenter",
  "address": "12 Elm Street",
  "inspection_date": "2021-04-22",
  "neighborhood": "Est",
  "district": "Montréal-Nord",
  "description": "bedbugs"
}

HTTP status code 201

The declaration was succesfully created.

Body

Media type: application/json

Type: object

Properties
  • declaration_num: required(integer)
  • first_name: required(string)
  • last_name: required(string)
  • address: required(string)
  • inspection_date: required(date-only)
  • neighborhood: required(string)
  • district: required(string)
  • description: required(union of string or nil)

Example:

{
  "id": 40885,
  "first_name": "John",
  "last_name": "Carpenter",
  "address": "12 Elm Street",
  "inspection_date": "2021-04-22",
  "neighborhood": "Est",
  "district": "Montréal-Nord",
  "description": "bedbugs"
}

HTTP status code 400

Validation error with submitted declaration.

HTTP status code 500

Error with the database query.

/api/declarations get

get

Return the declarations between two specified dates (inclusively).

get /api/declarations

Return the declarations between two specified dates (inclusively).

  • Request
  • Response

Query Parameters

  • du: required(date-only)

    Start date (ISO8601 format)

    Example:

    2018-06-13
  • au: required(date-only)

    End date (ISO8601 format)

    Example:

    2018-07-21

HTTP status code 200

Body

Media type: application/json

Type: array of object

Items: Declaration

  • declaration_num: required(integer)
  • declaration_date: required(string)
  • inspection_date: required(union of date-only or nil)
  • extermination_num: required(union of integer or nil)
  • treatment_start_date: required(union of date-only or nil)
  • treatment_end_date: required(union of date-only or nil)
  • neighborhood_num: required(string)
  • neighborhood_name: required(string)
  • district_name: required(string)
  • x_coord: required(union of number or nil)
  • y_coord: required(union of number or nil)
  • longitude: required(union of number or nil)
  • latitude: required(union of number or nil)

HTTP status code 400

One or many queryParameters is missing or invalid.

HTTP status code 500

Error with the database query.
